# ArduinoProjects
A collection of Arduino projects.

## SIMON SAYS
A memory game with 99 levels. It uses an LCD and an infrared remote as a controller.

PARTS:
- LCD (display)
- 3 LEDs
- Potentiometer (10kΩ)
- IR Receiver
- Resistors (330Ω)
- Arduino Uno R3
